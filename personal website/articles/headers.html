there is a video at the bottom of the page explaining the article, so if the article is confusing, you can wach that
in html there are tags that make a heading, these tags are: h1, h2, h3, h4, h5, and h6. what these tags make are things called headings.

in that order of tags(h1,h2,h3,h4,h5,h6) they get smaller the more numbers that get added to the end of the h, from h1, which makes very big text, to h6, which is slightly bigger then normal font size, and bolded.

you will usally only need to use h1,h2, and mabey h3, becuase the others are too small to be headings for something big, but the one you will use the most is by far h1, becuase it is the size of a usall heading for something big.

when to use what headings


you will use h1 tags for big things, to draw attention, but when will you need to use the other header tags?

i don't recommend using h4, h5, or h6, becuase they are too small, but what i do recommend are h3, h2, and h1

use h2 and h3 tags when there is already a h1 tag on the webpage, like on the many articles on this website, there is a big topic(in this article the big topic is headings), and sub topics(like in this article, a subtopic can be "what are headings?")a h2 tag and h1 tag looks a lot better then 2 h1 tags, or if you want, you can swap out h2 tags for h3 tags, if you want it to be smaller, but that's up to you
