meta tags


what is a meta tag?


meta tags represent the meta data of the website, such as; keywords so that whenever someone searchs for a word that's in the content property of the meta tag, your site can be listed on the rankings, and get viewed. another great meta tag is the description tag, whatever you put in the content property of this tag, will be displayed as a description of your website whenever someone searchs your site on a browser.

what's a meta tag's type?


a meta tag is a singular tag, meaning that you only need one of it to do something, unlike other tags such as; headings, links, paragraph, and most tags.


where is a meta tag usally kept?


meta tags are usally kept in the "head" tag, a tag where external data is supposed to be, such as description and keyword meta tags, or links to stylesheets for the webpage, or just setting up the charset.


examples of meta tags:


an example of the keyword tag is not something you can see, but it happens in the browser's code, where searching up a keyword, sites with that keyword in there meta tags, and sites that just say that word a lot will come, but you can not see it behind the broswer that you are on.

an example of a description tag on the other hand you can see, like when you search up a keyword in a browser, and a bunch of results show up, and under the links to the result, there will be a description of the website that the description is on.
