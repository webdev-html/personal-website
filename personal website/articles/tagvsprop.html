tags vs propertys

how are tags and properies different?

a tag is a value that does something, that is surounded by less then and more then signs. some highly used tags are: img tags, h1 tags, p tags, and many more.

a property on the other hand are values inside of tags that resemble a certian value for that tag, such as the property "src" for the img tag, in this case, the src propertie tells the img tag what's the web address of the thing you want to display as an image.

all properties resemble something inside a tag, and some properties can be in different tags, for the same reason, like how in an iframe tag, src is the property that tells the iframe what it should embed, the same way it does for images, so that shows that properties are not specific to a tag.

some highly used examples are: width for the width of something, height for the height of something, src for the directed path, and many more.
