spans


what is a span?


a span is an html element that is meant to be used as a container for other html elements.
a span does nothing without it having a child element.

what does a span do to it's child element?


what a span does to it's child element is that it makes every one of it's child elements on the same line, at first that might sound completely usless, but it makes elements the are always seperated be next to each other.
note: p tags bypass a spans effect

this will come in when you get into grids and flexbox, and for postitioning elements in your website.

fun fact: divs and spans are opposites, but they are used for the same reasons, postitioning elements on a webpage, but divs are used more often, because their purpose is needed more often. if you want to learn about divs, go to this link: article on divs
