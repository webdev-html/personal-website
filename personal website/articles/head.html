head tags


what are head tags?

head tags are the tags used for storing external data and meta data of the webpage, like keywords, the description of the website, viewport, and more(if you want to learn more about meta tags, i have an article on them like this one: meta tags article.)

by external data i mean linking files, like css stylesheets for the webpage.

what type is the head tag?

head tags are double tags, but every tag that's purpose is to have another tag placed in it(like head tags almost always being used to be the parent element to meta tags) will be a double tag, considering that a tag is different from a property.
